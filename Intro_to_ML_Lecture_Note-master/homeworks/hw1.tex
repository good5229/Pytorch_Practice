\paragraph{1.} When we use empirical cost to approximate the expected cost,

$$\mathbb{E}_{\vx \sim X} \left[ D(M^*(\vx), M, \vx) \right] \approx \frac{1}{N} \sum_{n=1}^N D(M^*(\vx^n), M, \vx^n)$$

is it okay to weigh each per-example cost equally? Given that we established that not every data $\vx$ is equally likely, is taking the sum of all per-example costs and dividing by $N$ reasonable? Should we weigh each per-example cost differently, depending on how likely each $\vx$ is? Justify your answer.

\vspace{5mm}

\paragraph{2.} A perceptron is defined as follows:
\begin{align*}
    M(\vx) = \sign(\vw^\top \vx + \vb),
\end{align*}
where $\vw \in \mathbb{R}^d, \vx \in \mathbb{R}^d$ and $\vb \in \mathbb{R}$. 
Why is the bias $\vb$ necessary? Provide an example where it is necessary.

\vspace{5mm}

\paragraph{3.} We used the following distance function for perceptron in the
lecture:
\begin{align*}
    D(M^*(\vx), M, \vx) = -\left( M^*(\vx) - M(\vx)
    \right) \left(\vw^\top \vx + \vb \right).
\end{align*}
This distance function has a problem of a trivial solution. What is the trivial
solution? Propose a solution to this.

\vspace{5mm}

\paragraph{4.} The distance function of logistic regression was defined as 
\begin{align*}
    D(y^*, \vw, \vx) = -(y^* \log M(\vx) + (1-y^*) \log (1- M(\vx))).
\end{align*}
Derive its gradient with respect to the weight vector $\vw$ step-by-step.
